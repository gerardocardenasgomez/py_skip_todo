# Py Skip Todo

Circular skip list todo CLI in Python:

- Infinite iteration through todos
- Visualize skip list structure in terminal
- Performance Benchmarks
- Unit Tests
- Simple web dashboard in Flask

# Supported Commands

- todo add <text>
- todo list
- todo done <id>
- todo search <id> | <string>
- todo visualize
- todo bench
- todo loop [--count N]
